#### 林業DX
##### 研究概要
日本の人工林資源は成熟し、本格的な利用期を迎えている一方で、林業従事者の減少が進み（昭和55年：14.6万人→令和2年：4.4万人）、森林調査業務の省力化・効率化が課題となっています。本研究では、この課題解決に向けて、森林画像情報を活用した森林調査業務の省力化・効率化を目的とし、深層学習に基づく森林画像認識システムの開発・評価に取り組んでいます。具体的には二つのシステムを開発しています。一つ目は、林内で撮影した360°画像から、森林資源を質と量の観点で評価するための森林物理量を推定する森林計測システムであり、目視に依存していた調査業務の効率化を図っています。二つ目は、ドローンなどのUAV（無人航空機）で取得したオルソ画像から樹種を判別する樹種判別システムであり、森林境界の明確化や所有地管理の効率化を図っています。これらの取り組みにより、森林調査業務の省力化を実現しました。具体的には、7haあたり3人で4日以上かかっていた森林調査業務を1人で1日でできるようになりました。

#### 歩容認証
##### 研究概要
ウェアラブルセンサーを用いた歩行認証は、継続的認証の有望なアプローチとして注目を集めています。しかし、既存の多くの研究では、制御された実験環境で収集された短期間の歩行データのみを用いて認証モデルを学習しており、日常生活における歩行パターンの多様性を十分に反映していない可能性があります。本研究では、インソール型歩行センサーを用いたディープラーニングベースの歩行認証モデルの学習データセットに、日常生活環境で収集された長期歩行データを組み込む効果について検討しました。歩行データは48名の健常参加者から、リアルタイムモード（短期データ用）と日常生活モード（長期データ用）の2種類の計測モードで収集しました。学習条件として、短期データのみを使用する条件と、短期データおよび長期データの両方を使用する条件の2つを比較しました。認証性能は、未知のユーザーに対して等誤り率（EER）を用いて評価しました。長期データを組み込むことで性能が向上し、平均EERは0.47%から0.17%へ、共通閾値EERは0.71%から0.28%へとそれぞれ低減しました。また、シルエットスコアおよびΔCosSimを用いた特徴空間分析により、テスト埋め込みの分離性が向上したことが示されました。さらに、SHAPに基づく分析から、ロール角関連パラメータへの依存度が低下し、つま先角度および最小足部クリアランスの寄与が増加したことが明らかになりました。
